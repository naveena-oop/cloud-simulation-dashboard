# Cloud Simulation Dashboard

## 📌 Overview
The Cloud Simulation Dashboard is a full-stack web application designed to simulate and monitor multi-cloud infrastructure environments. It provides a visual interface to compare and manage cloud services across different platforms.

---

## 🚀 Features
- Interactive dashboard for cloud monitoring  
- Simulation of multi-cloud environments  
- Visualization of cloud resource usage  
- Backend integration for handling data  
- Organized frontend and backend structure  

---

## 🛠️ Tech Stack
- Frontend: HTML, CSS, JavaScript  
- Backend: Node.js  
- Database/Services: Supabase
